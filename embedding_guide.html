Oversight AI - Embedding Guide

Overview

Oversight AI can be easily embedded into any website using iframes. We provide multiple embedding options to suit different use cases and space constraints.

Available Embedding Options

Version URL Recommended Size Best For
Full Interface / 800x600px or larger Dedicated pages, large sidebars
Embeddable /embed 600x500px or larger Blog posts, content pages
Minimal /embed/minimal 400x400px or larger Small widgets, mobile views

Basic Embedding

Standard Embed (Recommended)

<iframe src="http://localhost:12000/embed" width="600" height="500" frameborder="0" style="border-radius: 8px; box-shadow: 0 4px 15px rgba(0,0,0,0.1);"> </iframe>

Minimal Embed (For Small Spaces)

<iframe src="http://localhost:12000/embed/minimal" width="400" height="400" frameborder="0" style="border-radius: 8px; box-shadow: 0 4px 15px rgba(0,0,0,0.1);"> </iframe>

Responsive Embedding

CSS for Responsive Iframe

<style> .oversight-ai-container { position: relative; width: 100%; max-width: 800px; height: 0; padding-bottom: 75%; /* 4:3 Aspect Ratio */ margin: 20px auto; } .oversight-ai-iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; border-radius: 8px; box-shadow: 0 4px 15px rgba(0,0,0,0.1); } </style> <div class="oversight-ai-container"> <iframe class="oversight-ai-iframe" src="http://localhost:12000/embed"> </iframe> </div>

Advanced Configuration

Custom Styling

You can customize the appearance of the embedded widget by wrapping it in a container with custom CSS:

<div style=" border: 2px solid #667eea; border-radius: 12px; padding: 10px; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); box-shadow: 0 8px 25px rgba(0,0,0,0.2); "> <iframe src="http://localhost:12000/embed" width="600" height="500" frameborder="0" style="border-radius: 8px; width: 100%;"> </iframe> </div>

JavaScript Integration

For advanced integration, you can communicate with the embedded iframe using postMessage:

<script> // Listen for messages from the embedded Oversight AI window.addEventListener('message', function(event) { if (event.origin !== 'http://localhost:12000') return; // Handle messages from the embedded app if (event.data.type === 'analysis_complete') { console.log('Analysis completed:', event.data.sessionId); // Your custom handling here } }); // Send messages to the embedded app function sendTopicToOversightAI(topic) { const iframe = document.getElementById('oversight-ai-iframe'); iframe.contentWindow.postMessage({ type: 'set_topic', topic: topic }, 'http://localhost:12000'); } </script>

Security Considerations

Important: When embedding in production, make sure to:
  • Use HTTPS for both your site and the Oversight AI server
  • Configure proper CORS settings
  • Consider implementing Content Security Policy (CSP) headers
  • Validate that the iframe source is from a trusted domain

API Integration

For more advanced integrations, you can also use the REST API directly:

// Example API usage fetch('http://localhost:12000/api/analyze', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ topic: 'Artificial Intelligence', report_type: 'detailed' }) }) .then(response => response.json()) .then(data => { if (data.success) { console.log('Analysis completed:', data); // Handle the results } });

Available API Endpoints

Endpoint Method Description
/api/analyze POST Start a new analysis
/api/status/{session_id} GET Get processing status
/api/results/{session_id} GET Get analysis results
/api/download/{session_id} GET Download report as text file
/api/report-types GET Get available report types
Note: Replace http://localhost:12000 with your actual Oversight AI server URL when deploying to production.

Troubleshooting

Common Issues

  • Iframe not loading: Check CORS settings and ensure the server is running
  • Styling issues: Verify iframe dimensions and CSS conflicts
  • API errors: Check network connectivity and API endpoint URLs
  • Cross-origin issues: Ensure proper CORS configuration on the server
